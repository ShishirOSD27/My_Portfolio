# Premium Developer Portfolio

A high-end, professional portfolio website for Software and Android App Developers.

## 🚀 Features
- **Modern UI**: Glassmorphism, neon accents, and smooth animations.
- **Dynamic Content**: All data is managed through a single JSON-like file.
- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop.
- **Project Filtering**: Categorize and filter projects easily.
- **Interactive**: Typewriter effects, scroll reveals, and smooth carousels.

## 🛠️ Tech Stack
- HTML5 & CSS3
- Tailwind CSS (Play CDN)
- JavaScript (ES6 Modules)
- AOS (Animate On Scroll)
- Swiper.js (Carousels)
- FontAwesome (Icons)

## 📝 How to Update Content
You don't need to touch the HTML or CSS for basic updates. All content is located in:
`data/portfolioData.js`

Simply open that file and update the objects (Personal Info, Skills, Projects, etc.).

## 📸 Profile Image
Your profile image is located at `assets/profile.jpg`. To change it, replace this file with your own image keeping the same name, or update the path in `portfolioData.js`.

## 📤 Deployment
This portfolio is static and can be deployed for free on:
- **GitHub Pages**
- **Vercel**
- **Netlify**

Just upload the entire folder to your repository!
